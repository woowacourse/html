## 사용한 태그
1. div

   div는 플로우 콘텐츠를 위한 통용 컨테이너입니다. CSS로 꾸미기 전에는 콘텐츠나 레이아웃에 어떤 영향도 주지 않습니다

2. button
   button은 클릭 가능한 버튼을 나타냅니다. 버튼은 양식 내부는 물론 간단한 표준 버튼 기능이 필요한 곳이라면 문서 어디에나 배치할 수 있습니다.

3. a

   a(앵커 요소)는 `href` 특성을 통해 다른 페이지나 같은 페이지의 어느 위치, 파일, 이메일 주소와 그 외 다른 URL로 연결할 수 있는 하이퍼링크를 만듭니다.

4. label

   label는 사용자 인터페이스 항목의 설명을 나타냅니다.

5. img

   img는 문서에 이미지를 넣습니다.

6. input

   input는 웹 기반 양식에서 사용자의 데이터를 받을 수 있는 대화형 컨트롤을 생성합니다.

7. textarea

   textarea는 멀티라인 일반 텍스트 편집 컨트롤을 나타냅니다.

8. var

   var는 수학 표현 또는 프로그래밍에서 변수의 이름을 나타냅니다.

9. em

   em는 텍스트의 강세를 나타냅니다. em요소를 중첩하면 더 큰 강세를 뜻하게 됩니다.

10. audio

    audio는 문서에 소리 콘텐츠를 포함할 때 사용합니다.
