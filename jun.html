HTML의 역사

스위스의 유럽 입자 물리 연구소 CERN에서 1990년 WEB이라는 서비스와 HTML이 등장함(by 팀 버너스리)

CERN - 특성상 국가의 범위를 넘어서는 연구를 위해 만들어진 곳, 연구자들이 많은 정보와 자료를 주고 받음.
팀 버너스리가 효율적으로 정보를 주고 받을 수 있는 시스템을 1980년도부터 약 10년정도 연구하다가 1990년도에 결과물을 드러냄

  • 1990년 10월 - 웹페이지 에디터를 만듦
  • 1990년 11월 - HTML 페이지를 볼 수 있는 최초의 웹브라우저 World Wide Web을 만듦
  • 1990년 12월 - WEB 서버를 만들고, info.cern.ch라는 주소를 부여

HTML의 성장과 발전

HTML에 CSS와 JavaScript라는 언어가 함께하면서, 다음과 같이 발전됨

  1. 정적인 컨텐츠를 보여주던 문서
  2. 게시판에 글쓰기/수정하기/공유하는 웹 사이트
  3. 쇼핑/다양한 상호작용하는 웹 서비스 or 웹앱

HTML / CSS / JavaScript의 역할

  • HTML - 정보의 골격
  • CSS - 정보를 꾸며줌
  • Javascript - 동작하게 해줌

HTML이란

HyperText Markup Language의 약자
즉, 하이퍼텍스트를 중요한 특징으로 갖는 마크업 형식의 언어

용어 설명) 하이퍼텍스트 = 링크 / 마크업 언어 = 문법적인 특성을 가지고 있는 언어

HTML이 정보를 표현하기 위한 방법.

  • 엘리먼트 : <h1>I am Jamie</h1>
  • 여는태그 : <h1>
  • 컨텐츠 : I am Jamie
  • 닫는태그 : </h1>
