버전 공개일 내용
HTML 1.0 1991년 최초의 HTML로 팀 버너스 리가 월드 와이드 웹을 발표하면서 내놓은 버전
처음에는 버전이 붙지 않았으나 나중에 보강된 2.0 버전이 나오면서 이름이 붙음
80년대에 존재하던 SGML이라는 마크업 언어를 참조하여 만들어짐
HTML 2.0 1995년 11월 24일 HTML 사상 최초로 표준으로 지정 - 팀 버너스 리와 여러 다른 사람의 노력으로 표준화 됨
HTML 1.0에서 파일 업로드 양식과 프레임, 테이블, 이미지맵, 국제화 기능이 추가
인터넷의 대중화가 시작되면서 이때부터 HTML도 널리 알려지기 시작함
HTML의 문제점이 두드러졌던 시기 - 95년대 브라우저 전쟁 시기 웹페이지 관리자는 IE를 위한 페이지와 넷스케이프를 위한 페이지를 따로 만들어야 했었음 => W3C에서 보완한 것이 HTML 3.2
HTML 3.2 1997년 1월 14일 표준화 작업을 담당하는 W3C에서 처음으로 나옴
수학 수식을 사용하는 태그를 완전히 제외하고, 넷스케이프의 비주얼 관련 태그를 수록
<b>나 <font> 태그가 들어간 것이 해당 버전
HTML 4.0 1997년 12월 Strict, Transitional, Frameset의 세 가지 문서 형태를 지원하는 것이 가장 큰 변화
Strict - 비표준이나 비권장 태그를 절대 허용하지 않는 엄격한 문서
Transitional - 비표준이나 비권장 태그도 허용하는 융통성 있는 문서
Frameset - 웹브라우저 화면을 나눈 프레임 문서에 쓰는 것
HTML 4.01 1999년 12월 2014년 10월 28일 HTML5의 최종 권고안이 확정되면서 구버전이 됨
비주얼 태그가 모두 비권장으로 지정된 것이 가장 큰 변화 - 기존 비주얼 태그는 CSS로 빼서 사용할 것을 권장
XHTML 1.0 2000년 1월 26일 내용상의 변화는 거의 없고, HTML 4.01을 XML 형식으로 포팅한 버전.
HTML 4.01의 내용을 거의 그대로 가지고 있음
XHTML 1.1 2001년 5월 31일 XHTML 1.0까지 있었던 Transitional 형식이 빠지면서 비표준이나 비권장 태그와의 호환성이 사라지며 지나치게 엄격하다는 지적과 함께 사용되지 않게 됨
XHTML 2.0 2009년 말에 논의가 중단된 XHTML의 버전
원래 XHTML 1.1을 잇는 차기버전으로 이야기가 되고 있었지만 2008년 HTML 5로 방향을 선회하면서 중단
HTML 5 2014년 10월 28일 HTML의 최신 버전
HTML 5.1 2nd Edition 2017년 10월 3일 이름은 2nd Edition이지만, HTML 5의 차기작이라는 의미
HTML 5.1 1st Edition은 없다는 의미
HTML 5.2 2017년 12월 14일 HTML의 최신 마이너 버전(2013년 기준)
