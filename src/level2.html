2단계 : 태그 더 알아보기

2단계에 오신것을 환영합니다.

이번 단계에서는 몇가지 태그를 더 알아보겠습니다. 

<html>
  <head>
    <meta charset="UTF-8">
    <title>Title</title>
  </head>
  <body>
    <h1>1단계 : 웹페이지에 글을 출력해보자</h1>
    <p>
      HTML 문서는 크게 head 태그와
      <font color="#ff7f50">'어떠한 태그?'</font>
      로 이루어져 있습니다.
    </p>
    <input type="text" placeholder="정답">
    <button type="button">제출</button>
  <body>
</html>

위의 코드는 level1 페이지의 html 을 간소화 한것입니다.
