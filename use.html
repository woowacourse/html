90년생 HTML

  1. Jun 코치님의 90년대 HTML 강의
  2. HTML의 발전과 버전
  3. 해당 문서에서 사용한 HTML 태그 목록과 설명
  4. 참고자료
  1. <!DOCTYPE html> - DOCTYPE, 문서에 대한 버전 정의
  2. <html lang=""></html> - HTML문서의 루트에 해당하며, 문서의 기본 요소
  3. <head></head> - 문서에 사용되는 메타데이터 집합
  4. <meta charset=""> - title, base, link, style, script 요소로 표현할수 없는 다양한 종류의 메타데이터를 표현
  5. <title></title> - HTML문서의 타이틀을 지정, 브라우저의 타이틀바에 표시
  6. <body></body> - 문서의 메인 컨텐트
  7. <header></header> - 네비게이션을 돕거나 소개를 나타내는 그룹을 표현
  8. <nav></nav> - 다른 페이지나 같은 페이지 안에 다른 부분으로 이어주는 네비게이션 링크로 구성된 섹션을 표현
  9. <h1></h1><h2></h2><h3></h3> - 각 섹션의 제목을 나타냄
  10. <ol></ol> - 순서가 있는 목록을 나타냄
  11. <ul></ul> - 아이템의 순서가 중요하지 않은 목록을 나타냄
  12. <li></li> - 목록 아이템을 나타냄
  13. <a href="" target=""></a> - 텍스트에 링크를 설정, href : 링크 주소를 지정, target : 링크를 어떤 프레임에 열것인지를 결정
  14. <img/> - 이미지 삽입
  15. <br/> - 한 줄 띄우기
  16. <p></p> - 문단설정
  17. <section></section> - 문서나 응용프로그램의 일반적인 섹션을 표현
  18. <article></article> - 문서내에서 독립적인 컨텐츠를 나타냄
  19. <table></table> - 1차원 이상의 데이터를 표의 형태로 나타냄
  20. <thead></thead> - 테이블의 헤더행을 그룹화 하기 위해 지정
  21. <th></th> - 제목 셀 정의
  22. <tbody></tbody> - 데이터행을 그룹화하기 위해 지정
  23. <tr></tr> - 행 정의
  24. <td colspan=""></td> - 데이터 셀 정의, colspan : 해당 셀이 미치는 범위의 열 수를 지정
  25. <tfoot></tfoot> - 테이블의 푸터행을 그룹화하기 위한 요소
  26. <iframe></iframe> - 문서내에 다른 문서를 내포할 수 있게 해줌
  27. <footer></footer> - 주로 저작권 정보나 서비스 제공자 정보등을 나타냄
