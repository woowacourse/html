JavaScript의 탄생

페이지 이동하기

  • HTML 페이지
  • CSS 페이지
  • JavaScript 페이지

CSS는 페이지를 예쁘게 꾸밀 수 있게 해주었지만 사람들은 거기서 만족하지 못했습니다.
사람들은 페이지에 여러가지 동작을 원하기 시작했고 이를 충족시키기 위해 JavaScript가 등장했습니다.

JavaScript를 이용하면
옆의 버튼처럼 사이트의 전체적인 색을 변화시킬 수도 있습니다.
옆의 텍스트 입력창처럼 입력에 대해 반응할 수도 있습니다.

이러한 기본적인 움직임 말고도 우리가 실제 인터넷에서 보는 다양한 움직임을 추가할 수 있습니다.
